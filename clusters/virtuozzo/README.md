# Cluster: virtuozzo

This directory contains the Flux configuration for the Virtuozzo Kubernetes cluster.

## Structure

| File | Purpose |
|---|---|
| `flux-system/` | Auto-generated by `flux bootstrap` — do not edit manually |
| `infrastructure-kustomization.yaml` | Flux Kustomization reconciling `./infrastructure/` |
| `apps-kustomization.yaml` | Flux Kustomization reconciling `./apps/` (depends on infra, enables SOPS decryption) |

## Bootstrap

Run the bootstrap script from the repo root:

```bash
./scripts/02-bootstrap-flux.sh
```

This auto-generates the `flux-system/` directory with all Flux controllers and the
GitRepository source pointing back to this repository.
